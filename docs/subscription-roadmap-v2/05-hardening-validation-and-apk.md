# 05 集成加固、全量验证与最终 APK

## 目标

本阶段合并旧 06 和旧 07，不再增加主要产品功能。完成异常矩阵、生命周期、多账号、导入导出、性能、全量回归和最终 APK 放行。

## 开始门禁

- 阶段 01～04 均已有可安装 APK 和阶段报告。
- 真实账号验证中发现的问题已修复或进入书面已知问题清单。
- 冻结主要功能范围；只接受缺陷修复、测试、日志和必要兼容改动。

## 生命周期与并发审计

- 每个网络任务都携带账号、签名、generation，并在数据库/UI 提交前校验。
- 配置变化后最多一个共享更新任务；页面重建只重新订阅。
- Scene/View 销毁后解除 EventBus、LiveData/Flow、回调和倒计时观察。
- 协调器不持有 Activity、View 或短生命周期 Context。
- 退出、切换账号和有效查询变化使旧任务立即失效。
- 主线程只做轻量 HashSet 查询和绘制；SQLite、解析、哈希和大集合构造在后台。

## 导入导出

- 导出前执行 WAL checkpoint 或使用数据库提供的安全一致性快照方式。
- v8 导出包含三张新表，往返导入后镜像、检查点和计数一致。
- v7 备份允许缺少新表；导入事务成功后调用阶段 01 的幂等建表入口。
- 不同账号数据保持原 `ACCOUNT_KEY`，不合并到当前账号。
- 导入在临时数据库中校验 schema/完整性后再替换；失败保留原库。
- 导入成功后清空并按当前账号重建内存快照。

## 异常矩阵

逐项固定行为并留测试/日志证据：

- 未登录：管理和服务端写操作提示登录，不发无效请求。
- 普通断网：保留旧镜像、计数和检查点。
- 限流/拒绝/服务不可用：立即停止，不形成连续重试。
- 登录页重定向/异常 HTML：解析失败，不能当合法空列表。
- 合法空订阅：清空当前账号镜像和对应展示。
- gdata 部分缺失/无效时间：整次更新不提交。
- 边界漫画删除、同秒多本：仍正确定位。
- 删除书签：清理全部签名检查点。
- 快速切页面/账号/开关：旧回调不覆盖当前状态。

## 可观测性与隐私

- Debug 记录任务 ID、脱敏账号、签名摘要、页数、条目数、停止原因、结果和耗时。
- 不记录 cookie、API key、完整隐私搜索词或完整账号 ID。
- 错误提示区分网络、登录过期、限流、解析失败、取消和四页上限。
- Release 降低详细日志，但保留必要错误分类。

## 性能放行

- 详情标签绑定无网络、无逐标签 SQL，匹配为 HashSet O(1)。
- 镜像替换和聚合提交各为单事务。
- 列表页串行且最多四页，gdata 每批最多 25。
- Decoration 不在每帧扫描数据库或完整数据源。
- `SearchQueryPolicy` 每次请求只运行一次。
- 无轮询、WorkManager 或隐藏定时网络任务。
- 500 个订阅标签和常规页面数据量下无明显主线程卡顿。

## 自动化测试矩阵

- 阶段 01～04 的全部单测。
- v7 fixture 升级 v8，旧表关键数据一致。
- v7 导入 v8、v8 导出再导入、故障回滚。
- 多账号、游客、账号切换和登录过期。
- 更新算法的首次基线、去重、同秒、删除、四页状态和原子回滚。
- 搜索策略入口覆盖、原文保持和签名稳定。
- Feed 四入口推进规则与列表/网格绘制逻辑。
- 多次旋转、重复打开详情、快速切换无重复任务或监听累积。

## 真机/真实账号验收

- 三种主题、列表/网格、字体缩放和旋转。
- 获取订阅镜像；详情在线/离线高亮。
- 订阅、取消、排除的成功和失败路径。
- 首次更新为 0，后续新增计数合理；无逐标签请求，最多四页。
- 中文开关覆盖主页、搜索、订阅、标签、上传者、书签和服务器收藏。
- 主页、聚合、单标签、书签边界标志正确。
- 前后台、锁屏、断网恢复、VPN/IP 切换。
- 如具备条件，两个账号交替登录确认不串数据。

```powershell
.\gradlew.bat app:testAppReleaseDebugUnitTest

```

## APK 放行

构建配置以实际 Gradle 文件为准并报告：flavor、build type、application ID、versionCode、versionName、签名类型。Debug 包应能与正式版共存，但不能宣称它能直接迁移正式版私有数据目录。

```powershell
Copy-Item app\build\outputs\apk\appRelease\debug\app-appRelease-debug.apk artifacts\ehviewer-subscription-final-debug.apk
Get-Item artifacts\ehviewer-subscription-final-debug.apk | Select-Object FullName,Length,LastWriteTime
Get-FileHash artifacts\ehviewer-subscription-final-debug.apk -Algorithm SHA256
```
